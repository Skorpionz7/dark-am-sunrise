# Code of Conduct

## 1. Be Respectful
Treat everyone with kindness. No harassment, discrimination, or personal attacks.

## 2. Be Helpful
Keep discussions constructive. Offer feedback that helps others learn and improve.

## 3. Be Inclusive
Welcome people of all backgrounds and skill levels. Assume good intentions.

## 4. Keep It Clean
No spam, disruptive behavior, or inappropriate content.

## 5. Follow the Rules
Respect project maintainers, licensing, and contribution guidelines.

---

By participating in this project, you agree to follow this Code of Conduct
